TEDDET
======

Texto Es Data <-> Data Es Texto

[Versión en español](README.es.md)

Esto es un proyecto para crear una herramienta para abrir ficheros de texto en forma de data y escribir ficheros data en ficheros de texto. 

Inicialmente la idea es trabajar con fichero separados por coma y ficheros de columnas de ancho fijo.

Este proyecto tiene dos partes:
- La primera parte es una libreria (libteddet) escrita en haxe, la idea es que se pueda usar desde python y desde csharp a partir de los mismo fuentes.
- La segunda parte usando la versión python de la libreria construir una interface gráfica para poder escribir leer y escribir ficheros de estos tipos.
